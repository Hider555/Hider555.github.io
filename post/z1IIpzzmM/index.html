虚拟环境和包管理

2019-03-05 1 min read # Python

1.创建虚拟环境

python - m venv 虚拟环境名

2.启动虚拟环境

win:

虚拟环境名\Scripts\activate.bat

mac/linux:

source 虚拟环境名/bin/activate

3.退出虚拟环境**deactivate**

4.pip管理包

pip install 包名

5.生成requirements.txt

pip freeze > requirements.txt

6.使用requirements.txt

pip install -r requirements.txt
